# MOL3022

## Backend:

#### Dependencies:
- Python3.7
- pip[10.0 - 19.0.2]
- virtualenv

#### Initial setup
```
cd backend/
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python run.py
```

## Frontend:

#### Dependencies:
- yarn (npm install -g yarn)

#### Initial setup
```
cd frontend/
yarn
yarn update
yarn start
```
